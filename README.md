# Nifty100 Financial Intelligence Platform

## Project Overview

This project is part of the Bluestock Internship Sprint 1 - Data Foundation.

The objective is to build an ETL pipeline that extracts data from multiple Excel datasets, validates data quality, loads the cleaned data into a MySQL database, and performs financial analysis.

## Tech Stack

- Python
- Pandas
- MySQL
- MySQL Workbench
- Jupyter Notebook
- Git & GitHub
